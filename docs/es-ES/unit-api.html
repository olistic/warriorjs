API de la Unidad

Puedes llamar métodos sobre una unidad para obtener información acerca de ella.

Estos son los métodos que tienes disponibles:

unit.isHostile():

Determina si la unidad es enemiga. Una unidad atrapada no es considerada enemiga.

Devuelve

(boolean): Si es una unidad enemiga o no.

unit.isFriendly():

Determina si la unidad es amiga.

Devuelve

(boolean): Si es una unidad amiga o no.

unit.isWarrior():

Determina si la unidad es el guerrero.

Aliases

unit.isPlayer()

Devuelve

(boolean): Si la unidad es el guerrero o no.

unit.isBound():

Determina si la unidad esta atrapada.

Devuelve

(boolean): Si la unidad esta atrapada o no.

unit.isUnderEffect(effect):

Determina si la unidad esta bajo el efecto dado.

Parámetros

effect (string): El nombre del efecto.

Devuelve

(boolean): Si la unidad esta bajo el efecto dado o no.
