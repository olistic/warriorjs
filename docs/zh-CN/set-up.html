安装

首先从 npm 中全局安装 WarriorJS开始 。

打开命令行并运行：

npm install --global @warriorjs/cli

运行npm命令前请确保您已经安装了Node.js >=8 推荐安装方法可以通过official installer进行安装。

游戏安装完成后，你可以在命令行终端运行warriorjs启动游戏。

warriorjs

就是这么简单！这将引导你创造你的勇士。给你的战士起一个合适的名字, 并选择一个新手塔。

完成上面的操作后, 您应该在项目中看到以下文件结构 (我们决定在色雷斯角斗士之后命名我们的勇士):

warriorjs
└── spartacus-beginner
    ├── Player.js
    └── README.md
  • spartacus-beginner 您的配置文件的目录。
  • Player.js 就像你的战士的大脑, 你会经常编辑这个文件。
  • README.md 包含当前级别的说明。

继续打开 README.md 以查看初级的说明。
